# 開発者ガイド

このプロジェクトへの貢献に関心を持っていただきありがとうございます。

## 開発フロー

1. **Issueの作成**: バグ報告や機能追加の提案はIssueを作成してください。
2. **Branchの作成**: `main` ブランチから新しいブランチを作成してください。
   - 命名規則: `feature/feature-name` または `fix/bug-name`
3. **実装**: コードを実装します。
4. **テスト**: 変更に対するテストを追加・実行してください。
   ```bash
   npm test
   ```
5. **Pull Request**: 実装が完了したらPull Requestを作成してください。

## コーディング規約

- **TypeScript**: 型定義を適切に行ってください。`any` の使用は極力避けてください。
- **React**: 関数コンポーネントとHooksを使用してください。
- **Lint/Format**: コミット前にLintを実行してください。
  ```bash
  npm run lint
  ```
