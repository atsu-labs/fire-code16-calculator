# Product Overview

消防法に基づく複合用途防火対象物の用途別面積を正確に計算し、専用部・共用部の按分処理を自動化するWebアプリケーション。

## Core Capabilities

- **階層的面積管理**: 地上階・地下階の階構成を管理し、各階に用途と面積を設定
- **多段階の共用部按分**: 階共用部、建物全体共用部、特定用途間共用部（グループ共用部）の3レベルで自動按分計算
- **法令準拠計算**: 消防法施行令別表第一に基づく用途分類と専用部分面積比による按分ロジック
- **リアルタイム集計**: 入力変更時に即座に用途別延べ面積を再計算・表示
- **累積丸め処理**: 丸め誤差を最小化し、合計値の一致を保証

## Target Use Cases

- **消防設備設計者**: 複合用途建物の用途別面積算出と消防設備の設置判定
- **建築確認審査**: 用途別延べ面積の検証と法令適合性チェック
- **設計事務所**: 複雑な共用部按分が必要な複合用途建物の面積計算

## Value Proposition

手計算では煩雑で誤りやすい複数段階の共用部按分を自動化し、消防法に準拠した正確な用途別面積を即座に提供。累積丸め処理により合計値の不一致を防ぎ、信頼性の高い計算結果を保証する。
