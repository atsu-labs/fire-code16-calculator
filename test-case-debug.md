# デバッグ用テストケース

## 入力データ
- 2階: ５項ロ 200 m²
- 1階: ４項 100 m²、５項ロ 100 m²、建物共用部 50 m²

## 期待される計算結果

### 建物共用部の按分
全体の専有面積 = 100 + 100 + 200 = 400 m²

1階の建物共用部 50 m² の按分：
- ４項（1階）: (100 / 400) × 50 = 12.5 m²
- ５項ロ（1階）: (100 / 400) × 50 = 12.5 m²
- ５項ロ（2階）: (200 / 400) × 50 = 25.0 m²

### 案分後の階別・用途別面積

#### 2階
- ５項ロ: 200 (専有) + 0 (階共用) + 25 (1階からの建物共用按分) + 0 (グループ共用) = 225.0 m²
- 階の合計: 225.0 m²

#### 1階
- ４項: 100 (専有) + 0 (階共用) + 12.5 (その階からの建物共用按分) + 0 (グループ共用) = 112.5 m²
- ５項ロ: 100 (専有) + 0 (階共用) + 12.5 (その階からの建物共用按分) + 0 (グループ共用) = 112.5 m²
- 階の合計: 225.0 m² ← **これが間違って250になっている**

### 建物全体の集計
- ４項: 100 (専有) + 0 (階共用) + 12.5 (建物共用) + 0 (グループ共用) = 112.5 m²
- ５項ロ: 300 (専有) + 0 (階共用) + 37.5 (建物共用) + 0 (グループ共用) = 337.5 m²
- 総合計: 450.0 m²

## 問題の分析

案分後の階別合計が **入力値の合計（250 m²）** になっているが、正しくは：
- 1階の入力合計: 100 + 100 + 50 = 250 m²
- 1階への按分合計: 112.5 + 112.5 = 225 m² （1階の建物共用部50m²のうち、25m²は2階に行く）

したがって、1階の案分後の合計は **225 m²** が正しい。
